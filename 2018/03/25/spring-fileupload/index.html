Spring-Day06-FileUpload

发表于 2018-03-25 | 分类于 jAVA框架

Spring 实现文件上传

1 建立一个项目拷贝Spring配置文件和导入对应的jar包(ioc aop mvc )
2 分别在web.xml中配置DispatcherServlet和在Spring配置文件中开启组件扫描和标注形式的mvc，配置ViewResolver  
3 在WEB-INF 下写一个上传页面并写一个控制器(写组件创建标注和映射请求的标注)其中有个方法可以调整上传文件的页面。  
4 填充 upload.jsp 中的内容
  必须是post请求   enctype="multipart/form-data"，必须有一个文件元素 file 
5 配置文件解析器和拷贝对应jar包

控制器代码如下：

@RequestMapping("/fileUpload.do")
    public String FileUpload(MultipartFile myFile,String username,HttpServletRequest request){
        String orgFile  = myFile.getOriginalFilename();
        System.out.println("源文件名:"+orgFile);
        //截取后缀名
        String suffix = orgFile.substring(orgFile.lastIndexOf("."), orgFile.length());
        String newFileName = username+"_"+System.currentTimeMillis()+Md5Util.md5Str(orgFile)+suffix;
        //获取要存放上传文件的路径
        String realPath = request.getServletContext().getRealPath("datas");
        System.out.println(realPath);

        File file = new File(realPath+"\\"+newFileName);
        try {
            myFile.transferTo(file);
        } catch (IllegalStateException | IOException e) {
            e.printStackTrace();
        }
        request.setAttribute("imgPath", "datas\\"+newFileName);
        return "show";
    }
